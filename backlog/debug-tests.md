# Debug et tests

**Statut** : ✅ Complété (v2.28.0)

## Description

Fonctionnalités de test pour faciliter le développement sans buzzers physiques.

## Fonctionnalités implémentées

- [x] **Ctrl+Click sur joueur en PREPARE simule PONG**
  - En état PREPARE, Ctrl+Click sur un joueur simule la réponse au PING
  - Permet de tester sans buzzers physiques connectés
  - Le joueur passe de "en attente" à "prêt"
