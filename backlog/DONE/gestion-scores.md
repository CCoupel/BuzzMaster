# Gestion des scores

**Statut** : ✅ Complété (v2.18.0)

## Description

Points d'équipe dissociés des points joueurs avec deux types de scores distincts.

## Fonctionnalités implémentées

- [x] **Points d'équipe dissociés des points joueurs**
  - Champ `TEAM_POINTS` sur les équipes
  - Score total = TEAM_POINTS + sum(player scores)
  - Clic sur header équipe = points à l'équipe
  - Clic sur ligne joueur = points au joueur
